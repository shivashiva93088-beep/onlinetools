# PDF to Word Converter Tool

A high-quality PDF to Word converter with professional UI, built for beginners with no coding experience.

## Features
- Convert PDF files to editable Word documents (DOCX)
- Preserves layout, text, tables, and images
- Clean, responsive, and professional interface
- Secure file processing (files deleted after conversion)
- Free to use within CloudConvert's daily limits
- Error handling for file size, type, and API limits
